Following docs/upgrade.md

1. k3s version

```shell
kubectl get nodes -o wide
NAME            STATUS   ROLES                  AGE    VERSION        INTERNAL-IP     EXTERNAL-IP   OS-IMAGE             KERNEL-VERSION      CONTAINER-RUNTIME
suvanjan-home   Ready    control-plane,master   187d   v1.34.7+k3s1   192.168.1.168   <none>        Ubuntu 24.04.4 LTS   6.8.0-110-generic   containerd://2.2.3-k3s1
```

2. Cordon

```shell
kubectl cordon suvanjan-home
```

3. Install new k3s version

```
curl -sfL https://get.k3s.io | \
  INSTALL_K3S_VERSION="v1.35.4+k3s1" \
  K3S_KUBECONFIG_MODE="644" \
  sh -s - \
    --disable=traefik \
    --disable=metrics-server \
    --disable=servicelb \
    --kubelet-arg config=/home/suvanjan/.k3s/kubelet.config
```

4. Status checks passed! Uncordon the node

```
kubectl uncordon suvanjan-home
```
